# Simple CRUD

A minimal HTML example demonstrating the key features of Polybase DB:

 - Authentication using [Polybase DB Auth](/guides/authentication)
 - [get](/guides/read-data#get-a-single-record) a record from Polybase DB
 - [create](/guides/write-data#creating-a-record) new record
 - [update](/guides/write-data#creating-a-record) existing record using `setName()` defined in the schema
 - [delete](/guides/delete-data) existing record using `del()` defined in the schema
